# 🌌 Personal AI – ChatBot

> A futuristic AI chatbot built using **Spring Boot + React + OpenRouter API**, featuring an animated aurora UI and full mobile responsiveness.

---

## 🚀 Live Demo

🔗 **Live App:** https://ai-chatbot-sfs7.onrender.com  

---

## ✨ Features

- 🤖 AI-powered chat using **OpenRouter API**
- 💬 Real-time conversational interface
- 🌈 Animated Aurora background
- 🧊 Glassmorphism UI design
- ✨ Particle canvas animation
- 📱 Fully responsive (Mobile + Desktop)
- ⚡ Typing indicator & smooth animations
- 🔐 Secure API key handling with environment variables
- 🚀 Deployed on Render

---

## 🛠 Tech Stack

### 🔹 Backend
- Java 17  
- Spring Boot  
- OkHttp  
- Jackson  

### 🔹 Frontend
- React (CDN)  
- TailwindCSS  
- Lucide Icons  

### 🔹 AI Integration
- OpenRouter Chat API  

### 🔹 Deployment
- Render  
- Docker  

---

## 🏗 Architecture

```
User (Browser)
      ↓
React UI
      ↓
Spring Boot API (/api/chat)
      ↓
OpenRouter AI
      ↓
AI Response → UI
```
